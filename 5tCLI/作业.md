#常用CLI命令教程
##一些常用的CLI命令
原本掌握的：

|命令|用法|
|:---:|:---:|
|pwd|查看当前的路径|
|cd|切换路径|
|ls|查看当前路径下的文件和文件夹|
|mkdir|建立目录|
|mv|移动文件或者重命名|
|rm|删除|
|clear|清屏|


掌握的新命令：

|命令|用法|
|:---:|:---:|
|file|查看文件属性|
|cat|查看文档内容|
|touch|创建文件|
|history|查看执行过的命令|

##VIM配置的第一版
```
set nocompatible        "设置vim和vi不兼容。
syntax on               "语法高亮
set number              "行号
set cursorline          "突出显示当行
set softtabstop=4       "tab缩进四格
set confirm             "处理未保存文件或只读文件，提示确认 
filetype plugin on      “允许插件  
```
